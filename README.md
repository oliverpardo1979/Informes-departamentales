# Informes departamentales

Repositorio liviano para compilar en Overleaf los informes departamentales del Centro Javeriano de Competitividad.

El archivo `main.tex` funciona como selector. Por defecto compila el Informe 02. Para compilar otro informe, cambie la línea activa:

```tex
\input{Paper/informe_02_remuneracion_departamentos}
%\input{Paper/informe_02_productividad_departamentos}
%\input{Paper/informe_04_productividad_remuneracion_departamentos}
```

Los documentos principales son:

- `Paper/informe_02_productividad_departamentos.tex`
- `Paper/informe_02_remuneracion_departamentos.tex`
- `Paper/informe_04_productividad_remuneracion_departamentos.tex`
